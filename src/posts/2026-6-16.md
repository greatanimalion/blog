# 前端技术问答对话记录

**用户:** 前端动画相关的库

**助手:** 前端动画库大致可以分为全能 JS 库、框架专用库、CSS 库和视觉/3D 库等几大类。为了帮你快速定位，我整理了一份常见的主流库清单：

| 库名 | 类型 | 核心特点 | 体积 | 适用场景 |
| --- | --- | --- | --- | --- |
| **GSAP** | 全能 JS | 性能极致，功能全面，生态强大，可动任何 JavaScript 能触及的元素 | 中等，可按需加载 | 复杂时间轴、跨浏览器兼容的各类大型项目，如品牌官网 |
| **Anime.js** | 轻量 JS | 轻量、强大、API 优雅简洁，同时支持 CSS、SVG 和 JS 对象 | 轻量 | 中小型项目，需要快速实现丰富动画效果，且不想引入庞大库的情况 |
| **Motion One** | 现代 JS | 基于原生 Web Animations API，极致轻量，追求极致性能的首选 | 极轻 (~3KB) | 性能敏感型项目，现代化浏览器环境，追求最佳性能 |
| **Three.js** | WebGL 3D | 浏览器 3D 渲染的王者，功能强大，社区活跃 | 庞大 | 3D 可视化、3D 模型展示、Web 小游戏 |
| **PixiJS** | Canvas 2D | 高性能 2D 渲染引擎，基于 WebGL 自动降级 Canvas，图形性能出众 | 中等 | 2D 游戏、粒子特效、富交互的可视化项目 |
| **Framer Motion** | React 专有 | React 社区标杆级动画库，声明式 API，与 React 生态深度集成，开发体验极佳 | 中等 | 现代 React SPA 项目，实现 UI 交互动效 |
| **VueUse Motion** | Vue 专有 | Vue 生态中 Framer Motion 的替代品，提供了类似的使用体验 | 中等 | Vue 3 项目，实现流畅的 UI 交互动画 |
| **AutoAnimate** | 框架无关 | 零配置为 DOM 元素的增删改自动添加平滑过渡动画 | 轻量 | 快速为列表、网格等动态内容添加入场/出场动画 |
| **Animate.css** | CSS 预设 | 提供海量预设好的 CSS 动画类名，如 `fadeIn`、`bounce` 等，开箱即用 | 轻量 | 快速添加入场、退出动画，无复杂逻辑需求的页面，对新手友好 |
| **Hover.css** | CSS 悬停 | 专注于鼠标悬停的动画特效 | 轻量 | 需要为按钮、链接等元素增加精致悬停交互的官网或 H5 页面 |
| **ScrollReveal** | 滚动 | 轻松实现元素滚动到视口时，以各种动画样式优雅出现的效果 | 轻量 | 滚动视差、滚动加载动画 |
| **Lottie-web** | 设计师友好 | 通过 JSON 文件无缝渲染 AE 动画，完美还原设计师的创意 | 中等 | 图标动效、角色动画、复杂图形动效等高度定制化动画 |
| **Wow.js** | 滚动 | 与 Animate.css 搭配，让元素在滚动时触发预设动画 | 轻量 | 快速实现滚动触发的 CSS 动画效果 |
| **Popmotion** | 实用工具 | 用于创建自然物理运动（如重力、弹簧）的动画库 | 轻量 | 需要模拟物理效果的交互，如拖拽、惯性滑动 |
| **Mo.js** | 图形创意 | 专注于为 Web 创建丰富的图形和爆炸等动态效果 | 中等 | 创意艺术网站、品牌宣传动画、Loading 特效 |
| **KUTE.js** | 轻量 JS | 专注于高性能和低资源消耗的核心动画引擎 | 极轻 | 对包体积和性能极其敏感的基础动画项目 |
| **Velocity.js** | JS 动画 | 老牌动画库，以高性能和 jQuery 的友好继承著称 | 轻量 | 旧项目维护，或在无需完整 GSAP 时需要一个可靠的高性能动画方案 |

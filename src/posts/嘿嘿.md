# Vue 3 组合式 API 入门11

Vue 3 引入了组合式 API（Composition API），提供了更灵活的代码组织方式。

<img src="/blog/vite.svg" alt="setup 函数" />
## 选项式 API

## setup 函数

`setup` 是组合式 API 的入口函数：

```vue
<script setup>
import { ref, onMounted } from 'vue'

const count = ref(0)

const increment = () => {
  count.value++
}

onMounted(() => {
  console.log('Component mounted')
})
</script>

<template>
  <button @click="increment">{{ count }}</button>
</template>
```

## 响应式 API

### ref

用于创建响应式的基本类型数据：

```javascript
import { ref } from 'vue'

const name = ref('John')
console.log(name.value) // 'John'
```

### reactive

用于创建响应式的对象：

```javascript
import { reactive } from 'vue'

const state = reactive({
  count: 0,
  name: 'Vue'
})

state.count++ // 自动触发更新
```

## 计算属性

```javascript
import { ref, computed } from 'vue'

const firstName = ref('John')
const lastName = ref('Doe')

const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})
```

## 生命周期钩子

组合式 API 提供了对应的生命周期钩子：

| 选项式 API | 组合式 API |
|------------|------------|
| beforeCreate | setup |
| created | setup |
| beforeMount | onBeforeMount |
| mounted | onMounted |
| beforeUpdate | onBeforeUpdate |
| updated | onUpdated |

## 总结

组合式 API 让我们能够更好地组织代码逻辑，提高代码复用性和可维护性。
